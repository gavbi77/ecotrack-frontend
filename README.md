# 🌱 EcoTrack Frontend

Interface web para monitoramento de pegada de carbono pessoal.

## 📋 Sobre o projeto

Frontend do EcoTrack desenvolvido em React. Permite que usuários se cadastrem, façam login, registrem atividades e acompanhem suas emissões de CO₂ mensais.

## 🚀 Tecnologias

- React
- React Router DOM
- Axios

## 📦 Instalação
```bash
# Clone o repositório
git clone https://github.com/gavbi77/ecotrack-frontend.git

# Entre na pasta
cd ecotrack-frontend

# Instale as dependências
npm install

# Rode o projeto
npm start
```

## 🔗 Páginas

| Rota | Descrição |
|------|-----------|
| /register | Cadastro de novo usuário |
| /login | Login com email e senha |
| /dashboard | Painel de emissões e registro de atividades |

## ⚙️ Configuração

O frontend conecta automaticamente com a API em `http://localhost:3000`.

Certifique-se de que o backend está rodando antes de iniciar o frontend.

## 🖥️ Como usar

1. Acesse `http://localhost:3001`
2. Crie uma conta na tela de cadastro
3. Faça login com seu email e senha
4. No dashboard registre suas atividades diárias
5. Acompanhe o total de CO₂ emitido no mês
